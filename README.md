## About

This repository tracks vulnerabilities discovered and coordinated-disclosed by **1dayexploit**.

Each advisory contains:

- Affected product, vendor, and version range
- Root cause analysis of the vulnerability
- Impact assessment and CVSS vector
- Proof-of-concept, released post-patch per our disclosure policy
- Timeline of vendor communication
